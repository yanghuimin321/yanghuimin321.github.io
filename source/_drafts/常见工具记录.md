| VScode插件                          | 作用                                                         |
| :---------------------------------- | :----------------------------------------------------------- |
| **ESlint**                          | JavaScript/TypeScript代码的静态分析工具，用于发现和修改代码中的问题 |
| **Prettier**                        | 代码格式化工具，通过解析代码并重新按照统一的规则打印，确保代码风格一致性 |
| **GitLens**                         | Git增强插件，将Git功能深度集成到编辑器中Git增强插件，将Git功能深度集成到编辑器中 |
| **Git History**                     | Git 历史查看，可视化查看 Git 仓库历史记录                    |
| **Markdown Preview Enhanced (MPE)** | Markdown预览插件                                             |
| **Markdown All in One**             | Markdown 全能工具，提供完整的 Markdown 编辑功能套件          |
| **Paste Image**                     | 从剪贴板直接粘贴图片到 Markdown/编辑器中                     |
| **Auto Close Tag**                  | 自动闭合HTML/XML标签，当输入开标签时自动添加闭标签           |
| **Auto Import**                     | 自动导入模块、组件、函数、类型等，智能补全导入语句           |
| **Auto Rename Tag**                 | 自动重命名标签，修改HTML/XML的开/闭标签时，另一个同步修改    |
| **Color Highlight**                 | 颜色高亮，识别并高亮显示代码中的颜色值，直观展示颜色效果     |
| **Color Picker**                    | 颜色选择器，在代码中直接调出颜色选择器，可视化选择颜色       |
| **CSS Peek**                        | CSS 定义查看，快速查看和跳转到 CSS 类/ID 的定义              |
| **Image Preview**                   | 图片预览，在代码编辑器中直接预览图片                         |
| **Path Intellisense**               | 路径智能补全，文件路径自动补全                               |
| **Live Server**                     | 实时服务器，启动本地开发服务器，支持实时刷新                 |
| **Open in Browser**                 | 快速在默认浏览器中打开 HTML 文件                             |
| **WSL**                             | Windows Subsystem for Linux，在 Windows 上直接使用 Linux 开发环境 |
| **Vue**                             | Vue 3的官方语言服务工具，语法高亮、智能提示、类型检查、组件跳转 |
| **Vue VSCode Snippets**             | 提供大量Vue代码片段，快速生成Vue组件模板、指令、生命周期等   |
